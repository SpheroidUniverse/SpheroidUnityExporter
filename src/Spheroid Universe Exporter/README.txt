Sphroid Universe Exporter

Spheroid Universe Exporter plugin allows you to export a game scene created in Unity to 
Spheroid Script, in order to become a part of code created 
in the Spheroid Demiurge IDEÂ and then to be launched and played on mobile devices, 
both iOS and Android, in the XR Hub super-app. 

Designed to make the process of creating a scene easy and convenient for you. 
Use Unity to position, rotate and scale objects in the scene and animate it. 
Export your game scene to Spheroid Script and upload the resulting file to 
the Spheroid Demiurge IDE.
